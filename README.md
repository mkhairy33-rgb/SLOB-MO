# SLOB MRP Intelligence

Streamlit prototype for SAP/MRP-style SLOB optimization.

## What it analyzes
- Inventory value
- Unrestricted stock
- Allocated stock
- Blocked / QI stock
- Shelf life
- Forecast
- Demand
- Average consumption
- MOQ
- Lead time
- Customer count
- Material status

## Outputs
- Risk level per SKU
- Risk score
- Risk drivers
- Proposed action
- Owner suggestion
- Impact / effort ranking
- Ranked action list
- Executive control tower

Repository: your GitHub repo
